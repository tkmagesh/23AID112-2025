# Lab Practice (Python Programming)

## Question 1 — Array Operations

Write a Python program that:

1. Creates an array (Python list) of `n` integers entered by the user.
2. Inserts an element at a given position.
3. Deletes an element from a specified position.
4. Displays the final array.

**Input:**

* Number of elements
* Array elements
* Position for insertion
* Value to insert
* Position for deletion

## Question 2 — Linear Search

Write a Python program that:

1. Accepts `n` integers into an array.
2. Searches for a given element using **Linear Search**.
3. Prints the index if found, otherwise prints `"Element not found"`.

Also print the **number of comparisons performed**.
